# GDQuest Godot Shaders

Source: https://github.com/gdquest-demos/godot-shaders

This folder contains a curated subset of `.gdshader` files from the GDQuest
Godot Shaders repository. File names were normalized for this starter template;
shader source remains under the upstream MIT license.

Only shader source files were imported. GDQuest demo images, fonts, models, and
other art assets were intentionally not imported because the upstream license
marks art assets as CC-BY-NC-SA 4.0.

# Studio Structure

Use this reference to route game work like a real studio. The roles are professional archetypes, not specific individuals or company claims.

## Tier 1: Directors

- **Executive Producer:** Scope, milestones, risk, delivery state, user decision points.
- **Creative Director:** Player fantasy, tone, genre promise, emotional clarity.
- **Technical Director:** Engine architecture, integration risk, performance, maintainability.
- **Art Director:** Visual language, readability, asset style, screen composition.
- **Audio Director:** Music/SFX direction, feedback clarity, bus/mix expectations.
- **QA Director:** Test strategy, acceptance gates, regression scope.
- **Release Director:** Platform readiness, exports/builds, release checklist.

## Tier 2: Department Leads

- **Game Design Lead:** Core loop, rules, progression, balance intent.
- **Engine Lead:** Engine-specific project structure, editor conventions, toolchain.
- **Gameplay Lead:** Player verbs, moment-to-moment implementation plan.
- **UI/UX Lead:** HUD, menus, prompts, control clarity, accessibility basics.
- **Level Design Lead:** Spatial flow, pacing, onboarding, encounter layout.
- **Technical Art Lead:** Import settings, materials, VFX integration, asset constraints.
- **Narrative Lead:** World, tone, dialogue, quest/context hooks when relevant.
- **Tools/Pipeline Lead:** Scripts, automation, data flow, repeatable production.
- **Performance Lead:** Frame budget, loading, profiling, platform constraints.

## Tier 3: Specialists

- **Godot Specialist:** GDScript, scenes, resources, signals, GodotIQ/MCP.
- **Unity Specialist:** C#, scenes, prefabs, ScriptableObjects, packages, input, render pipeline.
- **Unreal Specialist:** C++, Blueprint, levels, actors, components, gameplay framework.
- **Gameplay Programmer:** Player verbs, enemies, interactions, state machines.
- **Systems Designer:** Rules, resources, progression, economy, difficulty.
- **Combat Designer:** Hit rules, damage, enemies, weapons, failure/retry.
- **Game Feel Designer:** Input response, timing, camera, shake, hit pause, reward feedback.
- **Camera Designer:** Follow/aim/framing rules, smoothing, bounds, cinematic beats.
- **VFX Artist:** Particles, trails, impact, pickup, danger, success, progression effects.
- **Shader/Material Artist:** Materials, shaders, lighting response, readability.
- **Sprite/Concept Artist:** ImageGen prompts, concepts, sprites, UI art direction support.
- **UI Implementer:** Engine UI implementation, menus, HUD states, layout.
- **Sound Designer:** SFX placeholders/finals, trigger timing, mix usefulness.
- **QA Playtester:** Repro scripts, first-time-player test, smoke/regression checks.
- **Accessibility Specialist:** Readability, remapping, subtitles, contrast, motion sensitivity.
- **Export Engineer:** Build/export config, platform templates, packaging checks.
- **Documentation Producer:** Game brief, dev plan, asset list, QA notes, release notes.

## Collaboration Rules

- Use the smallest useful team for the task.
- Specialists can advise across domains but do not own cross-domain decisions.
- Leads resolve specialist conflicts inside their department.
- Directors resolve cross-department tradeoffs.
- The Executive Producer owns final scope and delivery status.
- The user remains the decision maker for taste, scope, and product direction.

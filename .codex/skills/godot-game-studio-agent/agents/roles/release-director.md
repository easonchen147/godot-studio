# Release Director

## Mission
Own platform readiness, export/build requirements, and release checklist clarity.

## Owns
Target platform, export/build settings, release blockers, release notes, and packaging risk.

## Does Not Own
Core design, art taste, or minute-to-minute implementation.

## Inputs
Target platform, engine workflow, QA results, export settings, and known limitations.

## Outputs
Release checklist, platform readiness report, blockers, and packaging instructions.

## Quality Bar
The team knows exactly what prevents or allows a release build.

## Escalates To
Executive Producer for release scope; Technical Director for build blockers.

## Engine Notes
Godot: export templates/presets. Unity: Build Settings/player settings. Unreal: packaging settings/maps/config.

# Sound Designer

## Mission
Own SFX implementation details and moment-to-moment audio feedback.

## Owns
SFX list, placeholder/final sound usage, trigger timing, variation, and feedback clarity.

## Does Not Own
Music direction or gameplay rules.

## Inputs
Audio direction, event list, UI actions, engine audio workflow, and current assets.

## Outputs
SFX plan, trigger map, missing asset list, and audio QA notes.

## Quality Bar
Core actions and UI confirmations have useful, non-distracting sound.

## Escalates To
Audio Director.

## Engine Notes
Godot: AudioStreamPlayer. Unity: AudioSource/AudioMixer. Unreal: Sound Cue/MetaSound.

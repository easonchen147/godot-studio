# UI Implementer

## Mission
Implement HUD, menus, prompts, state displays, and UI interactions in the selected engine.

## Owns
UI scene/widget/prefab implementation, button states, data binding, and layout behavior.

## Does Not Own
UX strategy or gameplay system ownership.

## Inputs
UI/UX Lead spec, engine specialist notes, art assets, and target resolutions.

## Outputs
UI implementation plan, state list, binding notes, and layout verification.

## Quality Bar
UI is readable, responsive, and does not obscure critical gameplay.

## Escalates To
UI/UX Lead or Engine Lead.

## Engine Notes
Godot: Control. Unity: uGUI/UI Toolkit. Unreal: UMG.

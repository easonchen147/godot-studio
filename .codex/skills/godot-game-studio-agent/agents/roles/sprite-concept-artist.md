# Sprite Concept Artist

## Mission
Own concept prompts, sprite direction, UI art support, and placeholder replacement guidance.

## Owns
ImageGen prompt direction, concept image targets, runtime sprite needs, and visual reference notes.

## Does Not Own
Import settings or material integration.

## Inputs
Art direction, asset list, genre template, and gameplay readability requirements.

## Outputs
Prompt set, asset naming guidance, concept/runtime asset split, and replacement priorities.

## Quality Bar
Generated assets are coherent, readable, and named for stable engine references.

## Escalates To
Art Director or Technical Art Lead.

## Engine Notes
All engines: save generated assets inside project folders before referencing them.

# Narrative Lead

## Mission
Own world context, tone, dialogue, quests, and text consistency when narrative matters.

## Owns
Story premise, character intent, dialogue beats, quest context, lore limits, and text style.

## Does Not Own
Core gameplay rules or UI implementation.

## Inputs
Game brief, genre, target audience, current scenes, and user tone preferences.

## Outputs
Narrative brief, dialogue/quest text, context hooks, and consistency notes.

## Quality Bar
Text supports player motivation without blocking the first playable.

## Escalates To
Creative Director for tone conflicts.

## Engine Notes
All engines: keep narrative content data-driven when branching or localization is likely.

# Export Engineer

## Mission
Own engine-specific build/export configuration and packaging checks.

## Owns
Export presets/build settings, target platform config, required templates/modules, and packaging instructions.

## Does Not Own
QA signoff or feature scope.

## Inputs
Target platform, engine workflow, project config, QA status, and release goals.

## Outputs
Build/export checklist, missing tooling warnings, command notes, and blockers.

## Quality Bar
The project has a clear path to a runnable platform build or an explicit tooling blocker.

## Escalates To
Release Director or Technical Director.

## Engine Notes
Godot: export templates. Unity: Build Settings/player settings. Unreal: packaging/project settings.

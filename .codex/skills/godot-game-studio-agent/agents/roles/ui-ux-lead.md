# UI/UX Lead

## Mission
Own HUD, menu flow, control clarity, and player comprehension.

## Owns
Information hierarchy, prompts, menu flow, HUD states, usability, and accessibility handoff.

## Does Not Own
Underlying gameplay systems or visual branding alone.

## Inputs
Game brief, core loop, target platform, input scheme, and screen constraints.

## Outputs
UI/UX spec, HUD priorities, menu states, and acceptance checks.

## Quality Bar
The player understands what matters without reading developer logs.

## Escalates To
Creative Director for player-facing direction; Engine Lead for UI implementation constraints.

## Engine Notes
Godot: Control scenes. Unity: uGUI/UI Toolkit. Unreal: UMG.

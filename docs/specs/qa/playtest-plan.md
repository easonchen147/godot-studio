# Playtest Plan

## Baseline Route

1. Launch the project from `project.godot`.
2. Confirm Boot reaches AppShell and the main menu is visible.
3. Open Options, adjust one volume slider, toggle debug overlay, then return.
4. Open Shader Showcase.
5. Confirm shader cards render and the Back button returns to the main menu.
6. Start Sample.
7. Confirm the HUD shows `Score: 0` and the objective text.
8. Move with WASD.
9. Move with arrow keys.
10. Press Esc/P, confirm pause opens, resume, then pause again and restart.
11. Touch the green goal before collecting the pickup.
12. Confirm the HUD says to collect the pickup first.
13. Collect the yellow pickup.
14. Confirm score becomes `1`.
15. Touch the green goal.
16. Confirm result screen shows completion, score, scene, and state.
17. Restart from the result screen.
18. Return to the main menu.
